## Cypress Básico

* Exemplo de um projeto do curso básico de Cypress.

## Pré-requisito

* É necessário ter `Node.js` e `npm` instalados para executar este projeto.
* Utilizei as versões 16.13.0 do `Node.js` e 8.3.2 do `npm`. Recomendo que você use essas mesmas versões ou versões posteriores.

## Instalação
* Execute `npm install` (ou `npm i` para a versão abreviada) para instalar as dependências.

## Testes
* Você pode executar os testes simulando uma janela de visualização desktop ou mobile.

## Desktop
* Execute `npm test` (ou `npm t` para a versão curta) para executar o teste no modo janela de exibição.

* Execute `npm run cy:open` para abrir o Cypress no modo interativo da janela de exibição.

## Mobile
* Execute `npm run test:mobile` para executar o teste no modo janela do mobile.

* Execute `npm run cy:open:mobile` para abrir o Cypress no modo janela do mobile.
